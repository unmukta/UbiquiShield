Install for Google Chrome

Follow these quick steps to get Ubiqui Shield running on your browser.

1

Download the Extension

First, download the latest compiled version of the extension to your computer.

Download Chrome ZIP
2

Extract the ZIP File

Locate the downloaded UbiquiShield-Chrome.zip file in your downloads folder. Right-click on it and select Extract All... to unzip its contents into a folder.

3

Open Chrome Extensions

Open a new tab in Google Chrome and type chrome://extensions in the address bar, then press Enter.

4

Enable Developer Mode

In the top right corner of the Extensions page, locate the Developer mode toggle switch and turn it ON.

5

Load the Extension

Click the Load unpacked button that appears in the top left corner. Select the folder where you extracted the Ubiqui Shield files in Step 2.

You're all set! Ubiqui Shield is now active and protecting your privacy.
